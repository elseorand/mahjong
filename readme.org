* 仕様
	+ 開発者ツールでイカサマの発生はわからないようにする
	+ 一発リーチは、拾いで対応
	+ 積込はあり
	+ 防御は低コスト
	+ 攻撃は高コスト
	+ イカサマの基本は情報系にしたい
